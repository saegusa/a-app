# a-app

Web 向けの **将棋 CPU 対戦 MVP**（Angular v21（最新版想定））です。

## 現在の機能
- 9x9 盤面表示
- 通常初期配置
- 先手（プレイヤー） / 後手（CPU）のターン進行
- 合法手ベース移動（主要駒 + 成り）
- 持ち駒と打ち（同筋歩チェック）
- 王の捕獲で勝敗

## セットアップ
```bash
npm install
npm run start
```
